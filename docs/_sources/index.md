## About This Website
This website was built using Jupyter Book, a tool that transforms Jupyter Notebooks into a clean, navigable HTML site. Each page on this site is a fully executable Python notebook showcasing a different data analysis project completed as part of my coursework.

As you explore the projects, you will encounter a variety of Python and Pandas techniques including data cleaning with `pd.to_numeric()` and `coerce`, computing descriptive statistics such as mean, median, and mode, filtering and grouping DataFrames, and merging multiple datasets for cross-variable analysis. The projects also feature data visualization using both custom-built sparkline functions with the Python Standard Library and interactive charts created with Plotly Express. Each project applies these tools to real-world public datasets to investigate questions in public health, international policy, and social justice.
